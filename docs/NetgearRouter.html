NetgearRouter

NetgearRouter

Class representing a session with a Netgear router.

Constructor

new NetgearRouter(passwordopt, useropt, hostopt, portopt)

Properties:
Name Type Description
timeout number

http timeout in milliseconds. Defaults to 20000.
loggedIn boolean

login state.
Example
// create a router session, login to router, fetch attached devices
	const Netgear = require('netgear');

	const router = new Netgear();

	async function getDevices() {
		try {
			await router.login('myPassword');
			const deviceArray = await router.getAttachedDevices();
			console.log(deviceArray);
		} catch (error) {
			console.log(error);
		}
	}

	getDevices();
Parameters:
Name Type Attributes Default Description
password string <optional>
 'password'

The login password.
user string <optional>
 'admin'

The login username.
host string <optional>
 'routerlogin.net'

The url or ip address of the router. Leave empty to try autodiscovery.
port number <optional>

The SOAP port of the router. Leave empty to try autodiscovery.
